# Your Agent Control Center

## Stop Vibing. Start Directing.

Get AI agents working on your computer -- not chatbots, not coding tools. A workspace where conversation is the interface to learning, organizing, planning, and doing.

---

## Branding

- **Title:** Your Agent Control Center
- **The workspace:** Agent Control Center
- **The opposite:** Vibing
- **The pitch:** "Stop vibing. Start directing."

---

## Core Thesis

You don't need to learn to code. You don't need to become a prompt engineer. You need to learn how to think clearly and communicate what you want -- skills you already have. This workshop gets you set up with VS Code and Claude Code, and shows you what becomes possible when AI lives on your computer instead of in a browser tab.

---

## Format

**Workshop deck (Marp)** + **companion handout** for in-person delivery.

- **Duration:** ~30 minutes of instruction + installation time (~10-15 min buffer)
- **Audience:** Non-technical. Smart but new to AI agents.
- **Delivery:** Live, in-person. Attendees install during the workshop.
- **Deck:** `workshop-deck.md` (Marp, TBB dark luxury theme)
- **Handout:** `handout-best-practices.md` (take-home reference)
- **Troubleshooting:** `appendix-troubleshooting.md` (instructor reference + attendee fallback)

---

## Workshop Flow (~30 min + install buffer)

| Block | Time | Content | Deck Slides |
|-------|------|---------|-------------|
| Sell it | 3-5 min | What this is, the spectrum, you already know how | 1-4 |
| Install it | 10-15 min | VS Code, extension, sign in, first conversation | 5-10 |
| Use it | 10-15 min | Pick a real project, start talking, direct revisions, CLAUDE.md | 11-19 |
| Three rules | 3-5 min | Verify, one thing at a time, think before you ask | 20-22 |
| Close | 1-2 min | What you have now, what's next, stop vibing | 23-24 |
