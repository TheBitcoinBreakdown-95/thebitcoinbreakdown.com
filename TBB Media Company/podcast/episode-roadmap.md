## Launch Series (First 5 Episodes)

### EP01: The Golden Rules of Bitcoin (FIRST PODCAST)
- **Status:** Script near-complete (brain dump in `The Bitcoin Breakdown (old)/Lessons/Golden Rules Video/`)
- **Format:** Podcast (audio only, oratory delivery)
- **Length:** 15-25 min
- **Source:** Golden Rules Brain Dump.md, GR Script.md
- **Content:** 6 rules every bitcoiner lives by -- the culture, values, and survival instincts of the Bitcoin community
  1. Not Your Keys, Not Your Bitcoin (Mt. Gox, exchange risk, self-custody imperative)
  2. Don't Trust, Verify (assume everyone is a scammer, trustless protocols, DYOR)
  3. Start Small (DCA, don't go all-in as a beginner, conviction grows with knowledge)
  4. Stay Humble (never talk about your holdings, avoid greed/leverage/shitcoins, control emotions)
  5. Stack Sats and HODL (bitcoin is money not an investment, long-term thinking, generational wealth)
  6. Laser Eyes Until Fiat Dies (unwavering commitment, sound money standard, stay focused)
- **Teaching hook:** Open with: "Before you buy a single sat, you need to know the rules. Not the technical rules -- the cultural ones. The ones that keep bitcoiners alive."
- **Production steps:**
  1. [ ] Pull brain dump script from archive
  2. [ ] Edit into speakable outline (talking points, not prose)
  3. [ ] AI polish pass using Voice DNA profile
  4. [ ] User read-through, add riffs, cut what's wrong
  5. [ ] Record (OBS, one take, podcast audio)
  6. [ ] AI post-production: show notes, social posts, newsletter snippet
  7. [ ] Publish to Spotify for Podcasters
  8. [ ] Cross-post as TBB blog post with embedded audio

## Episode Production Checklist (Per Episode)

```
PRE-PRODUCTION
- [ ] Choose topic from roadmap
- [ ] Pull source material from content library
- [ ] AI drafts speakable outline using Voice DNA
- [ ] User edits: add riffs, cut wrong parts, mark emphasis points

RECORDING
- [ ] Set up OBS (audio-only for podcast, screen+audio for YouTube)
- [ ] Record (one take preferred, riff naturally off outline)
- [ ] Listen back for major issues (re-record only if necessary)

POST-PRODUCTION (AI-ASSISTED)
- [ ] Transcribe (Whisper)
- [ ] AI generates: show notes, 3-5 social posts, newsletter snippet
- [ ] User reviews and approves all generated content

PUBLISHING
- [ ] Upload to podcast host (Spotify for Podcasters)
- [ ] Schedule social posts (Buffer)
- [ ] Publish as TBB blog post with embedded audio (optional)
- [ ] Send newsletter (when subscriber base exists)
```
